# Sobre
Chatbot criado na plataforma BLIP, utilizando dotnet (C#). 

# Configurando sua aplicação
Navegue até https://portal.blip.ai e pegue o identificador de aplicação (application's identifier) e sua chave de acesso (access key).Para mais informações sobre como buildar a aplicação, acesse https://portal.blip.ai/#/docs.
